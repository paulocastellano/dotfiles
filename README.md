dotfiles
========

My personal dotfiles.

## Requeriments

* [asdf](https://asdf-vm.com)
* [git](https://git-scm.com)
* [vim](https://www.vim.org/)
* Curl
* [zsh](http://www.zsh.org)

## Instalation
```
git clone git@github.com:paulocastellano/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
chmod +x install.sh
./install.sh
```
